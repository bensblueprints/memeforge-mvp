## Show HN draft

**Title:** Show HN: Memeforge – a local desktop meme generator you buy once

**Body:**
I got tired of "Pro" tiers on meme/caption tools gating watermark-free export behind $16/month. So I built Memeforge — an Electron desktop app with a real canvas editor (drag/resize text boxes, auto-fit-to-width sizing, stroke-safe top/bottom bands) and a batch mode that reads a CSV and captions a whole folder of images unattended.

Two things I'm mildly proud of: the text auto-fit/word-wrap engine is a pure, dependency-free module (no browser font metrics — a documented glyph-width heuristic tuned for bold display caps) with a real unit-test suite, and the placeholder template pack is generated by a from-scratch PNG encoder (raw scanlines → Node's built-in zlib deflate → hand-assembled IHDR/IDAT/IEND chunks with CRC32) so there's zero image-library dependency in the whole app.

To be upfront: it doesn't ship licensed meme template art (Drake, etc.) — that's not mine to redistribute, and the placeholder pack is honestly just solid/gradient starter canvases. The real workflow is dropping in your own image.

Source is MIT on GitHub. $15 packaged installer for people who don't want to `npm i` — pay once, own it forever.

## AppSumo / PitchGround pitch

Memeforge is the anti-subscription meme/caption generator: a polished, dark-mode desktop app with a real drag/resize canvas editor, auto-fit-to-width text sizing, stroke-safe top/bottom bands, and a batch mode that captions an entire folder of images from a CSV in one unattended pass — with `.meme` project files so work is always reopenable, and zero bytes ever leave the user's machine. The caption/meme-tool category prints subscription revenue ($10–16/mo) on top of watermark and export-limit paywalls for what is, functionally, a canvas text tool — which makes a lifetime deal an easy sell: "Kapwing Pro costs $192/year; this is $15 once." MIT-licensed source doubles as trust and a technical differentiator (dependency-free layout engine, hand-rolled PNG encoder). Zero infrastructure cost per user means deep discount headroom for a launch campaign.
